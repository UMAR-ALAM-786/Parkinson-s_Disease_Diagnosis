# 🧠 Parkinson's Disease Diagnosis Using Machine Learning

> An AI-powered healthcare application for the early detection of Parkinson's Disease using Machine Learning and Streamlit.

## 📖 Overview

Parkinson's Disease is a progressive neurological disorder that affects movement, coordination, balance, and speech. Early diagnosis can help healthcare professionals provide timely treatment and improve patient outcomes.

This project utilizes Machine Learning algorithms trained on medical test and voice measurement data to predict the likelihood of Parkinson's Disease. The application is deployed using Streamlit, providing an easy-to-use and interactive web interface.

## 🩺 Common Symptoms of Parkinson's Disease

Some of the most common symptoms include:

- Tremors (shaking of hands, arms, legs, or jaw)
- Muscle stiffness and rigidity
- Slowed movement (Bradykinesia)
- Impaired balance and posture
- Difficulty walking
- Speech and voice changes
- Reduced facial expressions
- Handwriting changes
- Coordination problems
- Fatigue and sleep disturbances

## 🤖 How the Machine Learning Model Works

The model predicts Parkinson's Disease using medical test results and biomedical voice measurements.

### Step 1: Data Collection

Patient medical records and voice measurement datasets are collected for training.

### Step 2: Data Preprocessing

The data undergoes:

- Missing value handling
- Feature scaling
- Data cleaning
- Normalization

### Step 3: Model Training

Machine Learning algorithms learn patterns from historical patient data and identify characteristics associated with Parkinson's Disease.

### Step 4: Prediction

The user enters medical test values into the application.

The trained model:

- Analyzes the input values
- Compares them with learned patterns
- Predicts whether Parkinson's Disease is likely present

### Step 5: Result Generation

The prediction result is instantly displayed through the Streamlit web application.
